Getting Help

If you are stuck and need help on understanding the concepts or with using Alteryx or Tableau, there are two avenues for you to seek help:

1. Go to Knowledge: Post a question on Knowledge here. This is great If you need help on more detailed issues such as project help.

2. Chat live with students: Download the Slack application on your computer or phone and chat live with our student community here. Someone may be present and can help you immediately. Live chat is a great medium to talk about concepts you're learning.

Downloading From Links

There is a lot of data you you will need to download from our course in order to complete projects and quizzes. While this may sound simple there are a few common errors that can occur with this process.

  • Some links will download by left clicking the link
  • Some links will require you to right click the link and select save link as (This works best on a Google Chrome browser)
  • Some links are to google docs or sheets (If you are unable to access any resource on google drive please email in to bizanalyst-support@udacity.com)

Most issues occur when you left click a link, please try the right click save link as method if this occurs, if that does not work then please email in to bizanalyst-support@udacity.com
